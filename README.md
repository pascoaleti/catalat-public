# Catalat

Catalat is a catalog website platform for products, cars and real estate.

The project helps small businesses publish a fast public catalog, connect a custom domain and manage content from a private admin panel, without forcing the operation into a traditional ecommerce checkout.

## What Catalat offers

- Public catalog websites optimized for speed and SEO.
- Private admin panel for products, categories, pages and blog posts.
- Custom domain support with DNS and SSL automation.
- Multilingual public website and customer panel.
- Contact-first flow for businesses that sell through WhatsApp, phone, email or direct negotiation.

## Project status

Catalat is an active SaaS product.

The production source code is kept in a private repository. This public repository exists as an institutional and technical reference for the project.
