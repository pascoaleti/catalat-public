# Contributing

This repository is the public institutional profile for Catalat.

It is not the production SaaS source-code repository, so normal feature pull requests are not expected here.

## Good contributions here

- Documentation improvements.
- Product feedback.
- Public-facing typo corrections.
- Suggestions for clearer support, security or product information.

## Not accepted here

- Production SaaS source code.
- Credentials, tokens, database dumps or private customer data.
- Security exploit details in public issues.

For security reports, use [SECURITY.md](SECURITY.md).
